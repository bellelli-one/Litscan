# Лабораторная работа №2: Установка и настройка PostgreSQL. Основы работы с СУБД

## Цель
Создать простой веб-сервис на языке Go с использованием фреймворка Gin, шаблонов HTML.

1. **Установка PostgreSQL с использованием Docker-контейнера.**

2. **Запуск и базовое управление СУБД через терминальный клиент psql.**

3. **Создание пользователя базы данных и назначение прав.**

4.**Создание базы данных и таблиц с помощью SQL-команд.**

5.**Выполнение базовых CRUD-операций (вставка, выборка данных).**

6.**Подключение к серверу БД через графический IDE (на примере DBeaver).**
